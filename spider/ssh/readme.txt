1、在终端中输入以下命令：
    ssh-keygen -t rsa -b 4096 -C "pssh@spider.com"

-t rsa：指定密钥类型为 RSA。SSH 支持多种密钥类型，RSA 是最常用的一种。
-b 4096：指定密钥长度为 4096 位。更大的密钥长度通常更安全，但会增加计算开销。
-C "your_email@example.com"：为密钥添加注释，通常用来标识密钥的用途或拥有者。


2、运行命令后，系统会提示你选择保存密钥的路径和文件名，默认情况下，它会将密钥保存在 ~/.ssh/id_rsa（私钥）和 ~/.ssh/id_rsa.pub（公钥）：
你可以按 Enter 键使用默认路径，或者输入一个新的路径和文件名。

3、接下来，系统会提示你输入一个密码来保护私钥（可选）：
如果你希望为密钥设置密码，可以输入一个密码；如果不想设置密码，直接按 Enter 键。
